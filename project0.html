The Basics of Web and HTML

I. Web pages are composed with HTML. Understanding the basic syntax of HTML is needed to make a web page.

II. Using tags, we can insert images/links and change plain text into bold or italic.

III. The set of opening tag, content and closing tag is called an "element". There are inline and block elements. The block elements makes invisible boxes while inline don't.

IV. We can add style by using CSS. And the whole HTML document has a structure, which we would be learning in future lectures.
